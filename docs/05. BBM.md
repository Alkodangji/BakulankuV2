# PROJECT_CONTEXT_PART_05_BBM

# Modul BBM

## Tujuan

Modul BBM digunakan untuk mengelola data BBM, transaksi penjualan BBM, dan transaksi restok BBM.

---

# Struktur Modul

## Transaksi

Digunakan untuk melakukan penjualan BBM kepada pelanggan.

---

## Riwayat Penjualan

Menampilkan seluruh transaksi penjualan BBM.

Digunakan untuk:

* Melihat transaksi
* Edit transaksi
* Hapus transaksi

---

## Riwayat Restok

Menampilkan seluruh transaksi restok BBM.

Digunakan untuk:

* Melihat transaksi
* Edit transaksi
* Hapus transaksi

---

## Data BBM

Digunakan untuk mengelola data master BBM.

---

## Restok BBM

Digunakan untuk menambah stok BBM melalui transaksi pembelian.

---

# Data BBM

## Fungsi

Menyimpan informasi BBM yang dijual oleh toko.

Contoh:

* Pertalite
* Pertamax
* Solar

---

## Data

* Kode BBM
* Nama BBM
* Harga Beli
* Harga Jual
* Stok
* Stok Minimum

---

## Tambah BBM

### Validasi

1. Nama BBM wajib diisi
2. Harga Jual > 0
3. Stok >= 0

### Proses

1. Simpan data
2. Refresh tabel
3. Reset form

---

## Edit BBM

### Validasi

1. Data harus dipilih
2. Nama BBM wajib diisi
3. Harga Jual > 0
4. Stok >= 0

### Proses

1. Simpan perubahan
2. Refresh tabel
3. Refresh dashboard
4. Reset form

### Catatan

Perubahan stok melalui edit:

* Tidak dianggap transaksi
* Tidak mempengaruhi saldo
* Tidak membuat histori transaksi baru

---

## Hapus BBM

### Validasi

1. Data harus dipilih

### Proses

1. Konfirmasi
2. Hapus data
3. Refresh tabel

---

# Restok BBM

## Tujuan

Digunakan untuk mencatat pembelian BBM dari supplier.

---

## Data Input

* Tanggal
* BBM
* Jumlah Liter
* Total Harga
* Akun
* Catatan

---

## Perhitungan

Harga Beli:

Harga Beli = Total Harga / Jumlah Liter

---

## Validasi

1. BBM wajib dipilih
2. Liter > 0
3. Total Harga > 0
4. Akun wajib dipilih
5. Saldo akun mencukupi

---

## Proses

1. Hitung Harga Beli
2. Simpan transaksi restok
3. Tambah stok BBM
4. Hitung Harga Beli Rata-Rata
5. Update Harga Beli pada Data BBM
6. Kurangi saldo akun
7. Refresh Data BBM
8. Refresh Riwayat Restok
9. Refresh Dashboard
10. Reset form

---

# Harga Beli Rata-Rata

Harga beli pada Data BBM menggunakan metode rata-rata.

Rumus:

((Stok Lama × Harga Beli Lama) + (Stok Baru × Harga Beli Baru))
/
(Stok Lama + Stok Baru)

---

# Penjualan BBM

## Tujuan

Digunakan untuk mencatat penjualan BBM kepada pelanggan.

---

## Data Input

* Tanggal
* BBM
* Jumlah Liter
* Harga Jual
* Total
* Metode Bayar

---

## Perhitungan

Total = Liter × Harga Jual

---

## Validasi

1. BBM wajib dipilih
2. Liter > 0
3. Stok mencukupi

Jika metode bayar Cash:

4. Uang diterima >= Total

---

## Proses

1. Hitung total transaksi
2. Simpan transaksi
3. Kurangi stok BBM
4. Tambah saldo sesuai metode pembayaran
5. Refresh dashboard
6. Refresh riwayat
7. Reset form

# Catatan Modul

Harga beli pada histori restok tidak boleh berubah walaupun harga beli rata-rata pada Data BBM berubah.

Harga beli pada histori digunakan sebagai data historis transaksi.

Seluruh transaksi BBM harus menjaga konsistensi:

* Stok
* Saldo
* Harga Beli Rata-Rata
* Dashboard
* Laporan
